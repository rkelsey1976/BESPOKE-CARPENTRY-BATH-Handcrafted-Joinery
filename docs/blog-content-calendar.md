## On-Page Brief (Use for every post)

1. Use the exact primary keyword in:
   - SEO title
   - H1
   - First paragraph
   - One H2 (natural language)
2. Keep intros local:
   - Mention Bath (and nearby areas where relevant).
   - Reference common property context (period terraces, sash windows, damp-exposed joinery).
3. Add a practical conversion section:
   - "When to call a professional"
   - "Typical timeframe"
   - "Get a quote" CTA with phone and form links.
4. Internal links:
   - Minimum 2 service links
   - Minimum 1 area page link
   - Minimum 1 contact page link
